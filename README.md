## 开始使用

**首先，请确保您的设备满足以下推荐需求：**

- Windows 7 及以上版本的系统

> [!WARNING]
>
> 不确定SeewoKiller能否在Windows 7以下（不含）的Windows系统中运行，但有实验证明SeewoKiller可以在Windows XP上正常运行。如果您要在Windows 7(不含)以下的电脑上运行SeewoKiller，请直接下载Source Code而非setup.exe。

对于普通用户，可以在以下渠道下载到本软件，请根据自身网络环境选择合适的渠道。
