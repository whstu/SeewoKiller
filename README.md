## 功能
### 核心功能
<b>希沃锁屏破解</b>；晚自习制裁模式（附带日志功能）；一键卸载不必要的软件；去除希沃管家快捷启动栏；录制视频；一键防屏保；小游戏、恶搞和更多注册表功能
### 附加功能（免费，但可能需要另外下载）
<b>冰点还原破解</b>，计算π，AI（支持<i>Deepseek R1</i>和<i>Llama 3</i>）

## 开始使用

**首先，请确保您的设备满足以下需求：**

推荐配置：
- 处理器：Intel Core i3/i5 第 4 代或更高，或同等级 AMD 处理器
- 内存：8GB RAM 或更高
- 硬盘：固态硬盘 (SSD)，至少 100MB 可用空间
- 显卡：支持 DirectX 9 或更高，集成显卡即可
- 显示器：1366×768 分辨率或更高
- 输入设备：标准键盘和鼠标
- Windows 10/11 1903 或更高版本（64 位）
- 摄像头（录制视频时使用）

> 最低配置（能运行，但不保证能用）：
> - 处理器：Intel Pentium 4 2.0GHz 或同等级（支持 SSE2 指令集）
> - 内存：2GB RAM（运行 PyQt GUI 需要）
> - 硬盘：20MB 可用空间（机械硬盘亦可）
> - 显卡：支持基本图形显示
> - 显示器：800×600 分辨率
> - 输入设备：键盘（支持 WASD 导航）
> - Windows 7 SP1（32 位或 64 位）

> [!WARNING]
>
> 不确定 SeewoKiller 能否在 Windows 7 以下（不含）的 Windows 系统中运行，但有实验证明 SeewoKiller 可以在 Windows XP 上正常运行。如果您要在 Windows 7(不含) 以下的电脑上运行 SeewoKiller，请运行 fastboot 模式。
