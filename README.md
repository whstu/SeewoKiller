## 开始使用

**首先，请确保您的设备满足以下推荐需求：**

- Windows 7 及以上版本的系统

> [!WARNING]
>
> 不确定 SeewoKiller 能否在 Windows 7 以下（不含）的 Windows 系统中运行，但有实验证明 SeewoKiller 可以在 Windows XP 上正常运行。如果您要在 Windows 7(不含) 以下的电脑上运行 SeewoKiller，请直接下载 Source Code 而非 setup.exe。
